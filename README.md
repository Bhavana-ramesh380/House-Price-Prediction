# 🏠 House Price Prediction

A simple Machine Learning project that predicts house prices based on features like area, rooms, and location.  

Built to understand how real-world regression models actually work 🔍

---

## ⚙️ Tech Stack

- 🐍 Python  
- 📊 Pandas, NumPy  
- 🤖 Scikit-learn  
- 📓 Jupyter Notebook  
- 💾 Joblib  

---

## 📁 Project Files

- 📒 housepriceprediction.ipynb → model building + training  
- 📊 housing.csv → dataset  
- 💾 house_price_model.joblib → saved ML model  
- 📦 requirements.txt → dependencies  

---
🚀 How to Run in VS Code 

1️⃣ Open project
Open VS Code
Click File → Open Folder
Select House-Price-Prediction
2️⃣ Create & activate virtual environment
python -m venv venv

👉 Activate:

venv\Scripts\activate   # Windows
source venv/bin/activate  # Mac/Linux
3️⃣ Install dependencies
pip install -r requirements.txt
4️⃣ Run project
Open housepriceprediction.ipynb
Select kernel → Python environment
Click Run All
(Optional) Run .py file
python app.py


You can now train and test the model in VS Code.
