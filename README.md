# 🏠 House Price Prediction

A simple Machine Learning project that predicts house prices based on features like area, rooms, and location.  

Built to understand how real-world regression models actually work 🔍

---

## ⚙️ Tech Stack

- 🐍 Python  
- 📊 Pandas, NumPy  
- 🤖 Scikit-learn  
- 📓 Jupyter Notebook  
- 💾 Joblib  

---

## 📁 Project Files

- 📒 housepriceprediction.ipynb → model building + training  
- 📊 housing.csv → dataset  
- 💾 house_price_model.joblib → saved ML model  
- 📦 requirements.txt → dependencies
